# Temenos

## I. Temenos Interview Experience

The recruitment process at Temenos centers on strong database management skills and core problem-solving logic. Candidates are evaluated on their ability to write efficient queries and handle basic programming tasks.

#### 1. Aptitude Test
- **Format**: Potential initial screening round.
- **Focus**: Logical reasoning, quantitative ability, and verbal skills.

#### 2. Technical Assessment & Interview
- **SQL & Databases**:
    - High emphasis on writing queries using **Joins** (Inner, Left, Right).
    - Usage of **Aggregate Functions**: `MIN()`, `MAX()`, `SUM()`, `AVG()`, and `COUNT()`.
    - [Click & Solve: SQL Join Challenges](https://www.hackerrank.com/domains/sql)
- **Problem Solving & Coding**:
    - Focus on basic coding challenges that test logical flow rather than complex syntax.
    - **Language Specifics**: If you list a preferred language (Java, C++, Python), expect syntax-based questions or "how-to" implementation details.
    - [Click & Solve: Basic Logic Problems](https://leetcode.com/problemset/all/?difficulty=Easy)
